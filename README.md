# 🔒 Password Strength Checker

## DecodeLabs Cybersecurity Project 1 | Batch 2026

### 📌 Overview
This tool evaluates password security by checking length, character variety, common patterns, and calculating entropy.

### ✨ Features
- Length validation (8+ characters recommended)
- Character type detection (uppercase, lowercase, numbers, special)
- Common password blacklist
- Entropy calculation in bits
- Timing attack prevention demo
- Score from 0-10

### 🚀 How to Run
```bash
python password_checker.py
